An easy automated and advanced ticket builder for WordPress to capture tickets to one of your favourite Helpdesk.

== Description ==

WP Helpdesk Integration integrates WordPress with service desk tools to capture support tickets from your website. You can design the Form with default WordPress forms. The form created can be embedded in Post, Page or Widget. The information collected are sent to anyone of the helpdesk portal Freshdesk, Zendesk, Zoho Support & Vtiger Tickets module with secure API connection.

= Highlighted features =

* Captures support threads from WordPress to Freshdesk, Zendesk, Zoho Support and Vtiger Tickets module.
* Supports default WordPress Webforms.
* Redirects User to desired page after successful form submission.
* Notification on success or failure of Ticket capture.
* Embed forms in Page, Post or Widgets to capture Support threads.

== Installation ==

WP Helpdesk Integration can be installed in usual 3 methods

I. For simple general way to install

* Go to Plugins → Add New →Upload
* Browse and Upload the downloaded zip file
* Activate the plugin.

II. For familiar FTP users

* Extract the plugin zip
* Upload plugin zip to /wp-content/plugins/
* Go to Plugins → Installed Plugins → Active     

III. Straight from WordPress Admin

* Go to plugins → Add New
* Search for WP Helpdesk Integration
* Click Install Now → Activate
